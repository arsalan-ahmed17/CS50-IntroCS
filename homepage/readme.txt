CS50: Introduction to Computer Science - Problem Set 8: Homepage

This is a personal website project showcasing my hobbies, gallery, journey, and contact information. The website is designed to be mobile-responsive and includes a simple, clean, and user-friendly navigation bar.

Features:
Home: The main landing page of the website with a brief introduction.
Hobby: A page that discusses my hobbies.
Gallery: A gallery section to showcase images related to my interests.
My Journey: A section dedicated to my personal journey and experiences.
Contact: A contact form for visitors to reach out to me.
Mobile Responsiveness: The website adjusts its layout and design based on the screen size, offering a seamless experience on both desktop and mobile devices.


Tech Stack:
HTML: The basic structure of the website.
CSS: Styling for the webpage layout and design.
JavaScript: For interactivity such as the form submission and dynamic menu.
Bootstrap: A CSS framework to make the website responsive and provide pre-built components for styling.
